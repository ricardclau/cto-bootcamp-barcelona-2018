Students should have installed:

Git
Ansible (ideally 2.6.x with pip install ansible==2.6.7)
Vagrant 2.x (all tested with 2.2.0)
Virtualbox (5.1.18 works fine, tested with 5.2.20)
Packer latest (tested with 1.3.2)
Terraform 0.11.x (tested with 0.11.10, you can pick versions with tfenv at https://github.com/Zordrak/tfenv)
Docker latest (tested with 18.x)

In order to have vagrant boxes ready:
vagrant box add ubuntu/xenial64
